We find the best 3D structure for input by computing the sequence identity (using PSI-BLAST) of the input against all 3D structure complexes available from PDB and (selected) predicted by AlphaFold. The 3D structures are ranked in the decreasing order of sequence identity, and the one with the highest value is displayed by default. As a result, you may still see a 3D structure of input even if its 3D structure is unavailable in PDB.
