Coupling probabilites panel

The table here displays predicted probabilites of the given entries (both wild type sequences and/or mutations) by PRECOGx and known values from IUPHAR and the Shedding/ebBRET assays.

For rows displaying:

VAR = IUPHAR: primary coupling is represented by PC while the secondary ones as SC and the unknown ones as -.

VAR = LogRAi (Shedding experiment): values > -1.0 are coupling while others are not coupling (< -1.0) or unknown -.

VAR = Emax (ebBRET experiment): values > 0 are coupling while others are not coupling (= 0) or unknown -.

VAR = WT(Wild Type)/Mutation: values with predicted probability (by PRECOGx) greater than 0.5 are coupling while others are not coupling.

By clicking on any of the cells:

a) The corresponding input is shown in the PCA panel

b) The enriched/depleted contact positions of the input GPCR are highlighted in the Conacts panels and mapped onto a 3D structure in the Structure panel.

PCA panel

The PCA panel displays the projections of the human GPCRome sequences along with the first two principal components obtained by performing the Principal Component Analysis (PCA) on the embeddings of the human GPCRome for different layers.

The human GPCRome has a total of 377 receptors (279 Class A, 33 Class B, 17 class C, 11 class F, 25 Taste receptors and 14 in other classes).

Every input (WT/mutation) can be annotated with coupling information obtained from either Shedding, ebBRET, IUPHAR or STRING (only for B-arrs), or with class membership in IUPHAR.

Attention Map panel

Attention Map obtained from the most important attention head of the best performing embedding layer during the classification.

Contacts panel

This panel provides two options:

1. The same contacts are mapped to the input Sequence.

These contacts are either enriched (log-odds score > 0) or depeleted (log-odds score < 0).
Place the cursor over a selected position to view its naming in the GPCRdb generic numbering system.
Use the slider to set the absolute value cutoff of the Log-odds score. For example, a slider value of 0.3 will display the contacts that have an enriched (> 0.3) or depeleted (< 0.3) for the given G-potein coupling score.

2. Contacts Map displays the positions mediating the predicted contacts for the selected G-protein/B-arr group (from the table Coupling probabilties).

Choose between the different options using the dropdown.

Structure panel

This panel displays the predicted contacts mapped to the 3D structure of the input obtained from either PDB or AlphaFold. Zoom-in to view the GPCRdb generic numbering system of the selected positions.

Can't see any position? Try lowering the log-odds score threshold.
